# Everforest Theme

A comfortable and pleasant theme based on the [Everforest](https://github.com/sainnhe/everforest) color palette.

This repository contains ports of the Everforest theme for different editors.

## Available Versions

### [VS Code](./VSCode_Theme)

The VS Code version includes both Dark and Light variants with complete coverage for the editor, UI, terminal, and syntax highlighting.

[**View VS Code Instructions**](./VSCode_Theme/README.md) (Note: You may need to check the `VSCode_Theme` folder content if a specific README is missing there, otherwise see below for installation).

**Installation for VS Code:**
1. Copy the `VSCode_Theme` folder content to your extensions directory.
2. Or use `vsce package` inside the `VSCode_Theme` directory to generate a VSIX file.

### [Zed Editor](./Zed)

The Zed version includes Dark and Light variants optimized for Zed, along with recommended settings.

[**View Zed Instructions**](./Zed/README.md)

**Installation for Zed:**
1. Copy `Zed/everforest.json` to `~/.config/zed/themes/`.
2. (Optional) Apply settings from `Zed/zed_settings.json`.

## Features

- **Two variants**: Dark and Light themes
- **Carefully crafted colors**: Based on the official Everforest color scheme
- **Eye-friendly**: Warm colors that reduce eye strain
